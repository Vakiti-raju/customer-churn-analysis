# Customer Churn Analysis Dashboard

## Project Overview
This project analyzes customer churn behavior using Power BI to identify key factors influencing customer attrition and customer retention patterns.

## Objectives
- Analyze customer churn trends
- Identify high-risk customer segments
- Understand churn drivers
- Provide business insights for improving retention

## Tools & Technologies
- Power BI
- Data Modeling (DAX)
- Data Visualization
- Excel/CSV Dataset

## Key KPIs
- Total Customers
- Churned Customers
- Churn Rate (%)
- Revenue Impact

## Key Insights
- New customers experience the highest churn.
- Month-to-month contracts drive most churn cases.
- Higher monthly charges increase churn probability.
- Retention improves as customer tenure increases.

## Dashboard Features
- KPI Cards
- Risk Category Segmentation
- Tenure-Based Analysis
- Interactive Filters & Visualizations
